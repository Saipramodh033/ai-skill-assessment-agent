# Assessment Flow

1. User submits a job description and resume.
2. Backend creates a session.
3. Skill extraction identifies required, claimed, overlapping, and assessment target skills.
4. Frontend shows extracted skills for transparency.
5. Assessment engine asks one targeted question at a time.
6. Candidate answers are recorded and evaluated.
7. Gaps are identified from low or uncertain scores.
8. A learning plan is generated from gaps.
9. Final report shows readiness, evidence-backed skill scores, gaps, and next steps.
